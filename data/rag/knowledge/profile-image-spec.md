# プロフィール画像仕様

## 概要

この文書は架空プロダクト Workbench Profile のプロフィール画像変更仕様である。公開用のsynthetic knowledgeであり、実在サービスの内部仕様ではない。

## アップロード制約

### サイズと形式

プロフィール画像としてアップロードできるファイルサイズは最大5MBまでとする。対応形式はJPGとPNGのみである。

拡張子は `.jpg`、`.jpeg`、`.png` を受け付けるが、最終的な正当性はサーバー側の画像検証結果で判断する。

### 不正入力

未対応形式の場合は `unsupported_format` として扱う。5MBを超えるファイルは `file_too_large` として扱う。

## 更新後の表示

アップロード成功後は、レスポンスで返されたlatest profile image URLを画面状態へ反映する。ユーザーがページ全体を再読み込みしなくても、新しいプロフィール画像が表示されることを期待する。

## 受け入れ条件

- 5MB以下のJPGまたはPNGをアップロードできる。
- 5MBを超える画像は保存されず、ユーザーにサイズ超過エラーを表示する。
- JPG/PNG以外は保存されず、ユーザーに形式エラーを表示する。
- 成功時はlatest profile image URLを使ってプロフィール画像表示を更新する。
